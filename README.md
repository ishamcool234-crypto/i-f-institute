# I F Institute Website

A professional website designed for I F Institute, catering to students from Grade 1 to 11 (O/L).

## Features

- **Modern & Professional Design**: Uses a responsive layout with glassmorphism effects and smooth animations.
- **Comprehensive Sections**:
  - **Home**: Inspiring hero section with call-to-action buttons.
  - **About**: Highlight key features like expert faculty and proven results.
  - **Programs**: broken down into Primary (1-5), Junior Secondary (6-9), and O/L Prep (10-11).
  - **Registration**: A dedicated form to collect student details.
- **Student Registration Form**: Collects essential information including:
  - Student Full Name
  - Date of Birth
  - Grade Applying For (1-11)
  - Parent/Guardian Name
  - Contact Details (Phone, Email, Address)

## How to View

Simply open the `index.html` file in your preferred web browser (Chrome, Edge, Firefox, etc.).

## Customization

- **Colors**: You can change the primary/secondary colors in `style.css` under the `:root` variables.
- **Content**: Update text in `index.html` to reflect specific institute details.
- **Form Submission**: Currently, the form logs data to the console. To make it functional, you would need to connect it to a backend service or email API.
